### Step 2 · `chisha/refine_intent.py:RefineIntent.allows_methodology_break()`
新方法判断 refine 文本里有没有"破戒"信号:
```python
_METHODOLOGY_BREAK_KEYWORDS = [
    "破戒", "放纵", "放开吃", "今晚不管", "今晚就", "无所谓",
    "随便", "别管那么多", "一次性", "今天就这样",
]

def allows_methodology_break(self) -> bool:
    """L0-C 类硬契约是否被 refine 明确解除. 仅检查 raw_text + freeform_note,
    避免误判 — 必须有明确的"放纵 / 破戒 / 一次性"信号."""
    text = (self.raw_text or "") + " " + (self.freeform_note or "")
    return any(k in text for k in _METHODOLOGY_BREAK_KEYWORDS)
```
