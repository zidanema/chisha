# S-01 · sandbox-lab scaffold + 主题切换

**前置基础设施。后续 S-02/03 前端任务全依赖。**

参考: `docs/design_briefs/2026-05-19-sandbox-lab.md`,镜像 `apps/debug-ui/` 工程规范。

## What

创建 `apps/sandbox-lab/` Vite + React + TS 工程脚手架。

1. `apps/sandbox-lab/package.json` — name=chisha-sandbox-lab, port 5175。scripts: dev / build / preview / typecheck。deps: react@^18.3.1, react-dom@^18.3.1。devDeps: @types/react, @types/react-dom, @vitejs/plugin-react, typescript@^5.6.3, vite@^5.4.10 (与 debug-ui 完全对齐)。
2. `apps/sandbox-lab/vite.config.ts` — 端口 5175,proxy `/api → http://127.0.0.1:8765`,支持 `VITE_API_TARGET` 环境变量覆盖 (参考 apps/debug-ui/vite.config.ts)。
3. `apps/sandbox-lab/tsconfig.json` — 与 debug-ui 对齐 (strict, jsx: react-jsx)。debug-ui 不带 tsconfig.node.json,本任务也不建。
4. `apps/sandbox-lab/index.html` — 引入 Google Fonts (Inter, Noto Sans SC, JetBrains Mono) + root div。
5. `apps/sandbox-lab/src/main.tsx` — ReactDOM.createRoot 渲 App。
6. `apps/sandbox-lab/src/App.tsx` — 空 shell,显示一个 demo 区块验证字体 + accent + 主题。
7. `apps/sandbox-lab/src/styles.css` — 最小骨架: `:root` 定义 4 套 accent CSS variables (indigo/绿/玫红/琥珀,每套 accent/hover/soft/softer/ink/ring) + `body.theme-dark`/`body.theme-light` 主题切换。从原型 `chidiansha-sandbox/project/sandbox-lab/styles.css` 顶部 `:root` 块搬过来。
8. `apps/sandbox-lab/src/hooks/useTweaks.ts` — 简化版,只管 accent + theme,本任务先满足主题切换 demo。
9. `apps/sandbox-lab/README.md` — 仿 apps/debug-ui/README.md 起步说明 (≤ 50 行)。

## Why

S-02 视觉搬运 + S-03 交互骨架都依赖这个脚手架。本任务交付一个可以 `npm run dev` 起得来 + 主题/accent 切换可见的空壳,确保工程基线对齐项目规范。

## Done When

- `cd apps/sandbox-lab && npm install && npm run dev` → 5175 起得来,浏览器打开看到 demo 区块
- chrome-devtools-mcp 自驱验证: 切 accent (4 套) 整套 CSS variable 都变,切 dark/light 主题无遗漏硬编码色 (本任务范围内只有 demo 块)
- `npm run typecheck` 全绿
- console 无 error/warn,network 无 4xx/5xx
- `apps/sandbox-lab/` 与 `apps/debug-ui/` 同 layer,**互不依赖**(不 import 跨 app)

## Affected files

- 新建: `apps/sandbox-lab/{package.json, vite.config.ts, tsconfig.json, index.html, README.md, src/main.tsx, src/App.tsx, src/styles.css, src/hooks/useTweaks.ts}`
- 改: 无 (现有 chisha/* 和 apps/{debug-ui,web} 完全不动)
